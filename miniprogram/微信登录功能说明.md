# 微信登录功能说明

## 🎯 功能概述

本项目的微信登录功能支持三种模式，满足不同开发阶段和部署需求：

### 1️⃣ 本地模式（当前默认）
- **特点**：无需配置，开箱即用
- **适用**：开发测试阶段
- **限制**：生成模拟 openid，无法云端同步

### 2️⃣ 云开发模式（推荐）
- **特点**：微信官方提供，简单可靠
- **适用**：正式上线项目
- **优势**：免费额度充足，支持数据同步

### 3️⃣ 自定义服务器模式
- **特点**：完全控制，灵活扩展
- **适用**：企业应用，需要集成现有系统
- **要求**：需要自建后端服务

## 📱 用户使用流程

### 登录步骤
1. 打开小程序，点击底部"我的"标签
2. 如未登录，显示"未登录"状态
3. 点击"去登录"按钮
4. 进入登录页，显示"微信登录"按钮
5. 点击"微信登录"按钮
6. 弹出授权窗口，点击"允许"
7. 登录成功，显示用户昵称和头像
8. 自动返回首页

### 登录后功能
- ✅ 显示用户昵称和头像
- ✅ 查看基金统计数据
- ✅ 使用数据同步功能（云开发模式）
- ✅ 退出登录

## 🔧 当前配置状态

### 默认配置（app.js）
```javascript
globalData: {
  useCloudBase: false,  // ❌ 未启用云开发
  serverUrl: '',        // ❌ 未配置服务器
}
```

**当前模式：本地模式（测试）**

### 工作流程
```
用户点击登录
    ↓
wx.login() 获取 code
    ↓
生成模拟 openid: local_时间戳_随机字符
    ↓
wx.getUserProfile() 获取用户信息
    ↓
保存到本地存储
    ↓
登录完成（显示用户信息）
```

## 🚀 如何启用云开发模式

### 第一步：开通云开发
1. 打开微信开发者工具
2. 点击顶部"云开发"按钮
3. 开通云开发服务
4. 创建云环境（记录环境 ID）

### 第二步：上传云函数
1. 右键点击 `cloudfunctions/login` 文件夹
2. 选择"上传并部署：云端安装依赖"
3. 等待部署完成

### 第三步：修改配置
打开 `app.js`，修改以下内容：

```javascript
globalData: {
  useCloudBase: true,  // ✅ 启用云开发
  serverUrl: '',
},

// 在 initCloud() 方法中
initCloud() {
  wx.cloud.init({
    env: 'your-env-id',  // ✅ 填入你的环境 ID
    traceUser: true,
  });
}
```

### 第四步：测试
1. 重新编译小程序
2. 点击登录
3. 查看控制台，应该显示真实的 openid

## 🐛 常见问题

### 1. 点击登录无反应
**可能原因：**
- 网络问题
- 基础库版本过低
- 代码报错

**解决方法：**
- 检查网络连接
- 更新微信版本
- 查看控制台错误信息

### 2. 授权弹窗不显示
**可能原因：**
- `wx.getUserProfile` 需要在点击事件中调用
- 基础库版本 < 2.10.4

**解决方法：**
- 确保在按钮 `bindtap` 中调用
- 升级微信或调整基础库版本

### 3. 登录成功但显示"未登录"
**可能原因：**
- 数据未保存到本地
- 页面未刷新

**解决方法：**
```javascript
// 确保登录成功后调用
wx.setStorageSync('userInfo', userInfo);
app.globalData.userInfo = userInfo;
app.globalData.hasUserInfo = true;
```

### 4. 重启后登录状态丢失
**可能原因：**
- 未正确保存到本地存储

**解决方法：**
- 检查 `wx.setStorageSync` 是否成功
- 检查 `app.checkLoginStatus()` 是否正确读取
