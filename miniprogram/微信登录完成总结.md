## 🎯 使用方式

### 快速开始（本地模式）

1. **直接运行**
   ```bash
   # 用微信开发者工具打开 miniprogram 目录
   # 点击编译，即可使用
   ```

2. **测试登录**
   - 切换到"我的"页面
   - 点击"去登录"
   - 点击"微信登录"
   - 授权后即可看到用户信息

3. **当前配置**
   ```javascript
   // app.js（无需修改）
   globalData: {
     useCloudBase: false,  // 本地模式
     serverUrl: '',
   }
   ```

### 切换到云开发模式

1. **开通云开发**
   - 点击工具栏"云开发"
   - 创建云环境

2. **上传云函数**
   - 右键 `cloudfunctions/login`
   - 选择"上传并部署"

3. **修改配置**
   ```javascript
   // app.js
   globalData: {
     useCloudBase: true,  // ✅ 启用
     serverUrl: '',
   }
   
   initCloud() {
     wx.cloud.init({
       env: 'your-env-id',  // ✅ 填入环境ID
       traceUser: true,
     });
   }
   ```

### 切换到自定义服务器

1. **实现后端接口**
   ```
   POST /api/wx-login
   参数: { code }
   返回: { success, token, openid }
   ```

2. **修改配置**
   ```javascript
   // app.js
   globalData: {
     useCloudBase: false,
     serverUrl: 'https://your-server.com',  // ✅ 填入地址
   }
   ```

3. **配置域名白名单**
   - 在微信公众平台添加服务器域名

## 🎓 技术要点

### 1. 微信登录流程
```javascript
// 核心流程
wx.login() 
  → 获取 code
  → 换取 openid（三种模式）
  → wx.getUserProfile()
  → 保存用户信息
```

### 2. 状态管理
```javascript
// 全局状态
app.globalData.userInfo
app.globalData.openid
app.globalData.hasUserInfo

// 本地存储
wx.setStorageSync('userInfo', userInfo);
wx.setStorageSync('openid', openid);
wx.setStorageSync('token', token);
```

### 3. 模式切换
```javascript
// 通过配置控制
if (useCloudBase) {
  loginWithCloud();
} else if (serverUrl) {
  loginWithServer();
} else {
  loginLocalMode();
}
```
