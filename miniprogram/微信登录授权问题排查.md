# 微信登录授权问题排查指南 🔍

## 问题现象

登录后显示"微信用户"而不是真实的昵称和头像。

---

## 可能原因

### 1. wx.getUserProfile 没有被调用
**现象：** 
- 点击登录后没有弹出授权窗口
- 直接显示"登录成功"
- 用户信息显示为默认值

**原因：**
- 代码逻辑问题
- 在异步操作后调用
- 基础库版本问题

### 2. 使用了缓存的默认用户信息
**现象：**
- 之前测试时保存了默认用户信息
- 再次登录时直接读取缓存

**原因：**
- 本地存储中有旧数据

### 3. 真机环境限制
**现象：**
- 开发工具正常，真机不正常
- 或反之

**原因：**
- 环境差异
- 权限配置问题

---

## 解决方案

### 方案 1：清除登录信息重新登录 ⭐

#### 步骤：
1. 在"我的"页面点击"退出登录"
2. 清除所有缓存
3. 重新进入登录页
4. 点击"微信登录"
5. 注意观察是否弹出授权窗口

#### 代码层面清除：
```javascript
// 在登录页 onLoad 时自动清除无效数据
cleanInvalidUserInfo() {
  const userInfo = wx.getStorageSync('userInfo');
  if (userInfo && userInfo.nickName === '微信用户') {
    // 清除无效数据
    wx.removeStorageSync('userInfo');
    wx.removeStorageSync('token');
    wx.removeStorageSync('openid');
  }
}
```

### 方案 2：检查控制台日志

点击登录后，查看控制台输出：

#### 正常情况应该看到：
```
=== 开始微信登录流程 ===
即将调用 wx.getUserProfile...
✅ 获取用户信息成功: {nickName: "真实昵称", avatarUrl: "..."}
用户信息已保存: {...}
开始调用 wx.login 获取 openid...
✅ 微信登录成功
```

#### 异常情况：
```
=== 开始微信登录流程 ===
即将调用 wx.getUserProfile...
❌ 获取用户信息失败: {errMsg: "..."}
```

### 方案 3：使用"更新信息"功能

如果已经显示"微信用户"：

1. 在"我的"页面，用户信息右侧会显示"更新信息"按钮
2. 点击"更新信息"
3. 确认弹窗后会重新进入登录页
4. 再次授权获取用户信息

### 方案 4：检查项目配置

#### 检查 project.config.json
```json
{
  "libVersion": "2.33.0",  // 应该 >= 2.10.4
  "appid": "wxf3571e5676560ac1"  // 确保正确
}
```

#### 检查 app.json
```json
{
  "permission": {
    "scope.userInfo": {
      "desc": "用于完善用户资料"
    }
  }
}
```

### 方案 5：真机调试

#### 开发工具中：
1. 点击顶部"预览"或"真机调试"
2. 扫码在手机上打开
3. 点击登录，查看是否弹出授权窗口
4. 检查手机微信设置 → 通用 → 辅助功能 → 小程序 → 是否允许授权

## 常见错误

### 错误 1: "can only be invoked by user TAP gesture"
**原因：** 不在点击事件的同步代码中调用

**解决：**
```javascript
// ❌ 错误
async handleLogin() {
  await someOperation();
  wx.getUserProfile({...});  // 错误！
}

// ✅ 正确
handleLogin() {
  wx.getUserProfile({...});  // 直接调用
}
```

### 错误 2: "getUserProfile is not a function"
**原因：** 基础库版本过低

**解决：**
1. 更新微信到最新版本
2. 或在 project.config.json 中提高 libVersion

### 错误 3: 弹窗一闪而过
**原因：** 用户之前拒绝过授权，微信记住了

**解决：**
1. 微信设置 → 通用 → 发现页管理 → 小程序
2. 找到小程序 → 清除授权
3. 重新打开小程序

## 预防措施

### 1. 代码规范
```javascript
// 始终在点击事件中直接调用
handleLogin() {
  wx.getUserProfile({
    desc: '用于完善用户资料',
    success: (res) => {
      // 处理成功
    },
    fail: (err) => {
      // 处理失败
    }
  });
}
```

### 2. 数据验证
```javascript
// 保存前验证
if (userInfo.nickName && userInfo.nickName !== '微信用户') {
  wx.setStorageSync('userInfo', userInfo);
} else {
  console.warn('用户信息无效，不保存');
}
```

### 3. 添加日志
```javascript
// 关键步骤都加日志
console.log('步骤1: 开始登录');
console.log('步骤2: 获取用户信息成功', userInfo);
console.log('步骤3: 保存到本地');
```

## 仍然无法解决？

### 检查以下几点：

1. **AppID 是否正确**
   - 在 project.config.json 中检查
   - 是否使用了测试号

2. **小程序是否已发布**
   - 体验版或开发版可能有限制

3. **微信版本**
   - 确保微信版本 >= 8.0.0

4. **手机系统**
   - iOS 或 Android 是否有特殊限制

5. **网络环境**
   - 是否能正常访问微信服务器
