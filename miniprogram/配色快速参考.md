# 🎨 配色快速参考卡

## 核心色值速查

### 主色 - 金融蓝
```
#1677FF  核心蓝（主按钮、品牌色）
#0052D9  深蓝（渐变暗部）
#4096FF  浅蓝（渐变亮部）
```

### 涨跌色 - 中国标准（红涨绿跌）
```
#D32F2F  朱红（涨）
#388E3C  墨绿（跌）
```

### 预警色
```
#FFA726  琥珀色（警告文字）
#FFF3E0  浅橙色（警告背景）
```

### 文字色
```
#1A1A1A  主要文字
#666666  次要文字
#999999  辅助文字
#CCCCCC  禁用文字
```

### 背景色
```
#F5F5F5  页面背景（浅灰）
#FFFFFF  卡片背景（纯白）
#FAFAFA  数据区背景
```

### 分割线
```
#E0E0E0  主要分割线
#F0F0F0  次要分割线
```

## CSS 变量

```css
var(--finance-primary)        /* #1677FF */
var(--finance-rise)           /* #D32F2F */
var(--finance-fall)           /* #388E3C */
var(--finance-warning)        /* #FFA726 */
var(--finance-bg)             /* #F5F5F5 */
var(--finance-card-bg)        /* #FFFFFF */
var(--finance-text-primary)   /* #1A1A1A */
var(--finance-text-secondary) /* #666666 */
var(--finance-border)         /* #E0E0E0 */
```

## 常用工具类

```css
.rise              /* 涨色（朱红） */
.fall              /* 跌色（墨绿） */
.finance-number    /* 金融等宽字体 */
.warning-badge     /* 预警标签 */
```

---

## 渐变背景

```css
/* 页面背景 */
background: linear-gradient(180deg, #0052D9 0%, #1677FF 50%, #4096FF 100%);

/* 按钮渐变 */
background: linear-gradient(135deg, #1677FF 0%, #0052D9 100%);
```

## 图表渐变（Canvas）

### 涨势（朱红）
```javascript
gradient.addColorStop(0, 'rgba(211, 47, 47, 0.3)');
gradient.addColorStop(0.5, 'rgba(211, 47, 47, 0.1)');
gradient.addColorStop(1, 'rgba(211, 47, 47, 0)');
```

### 跌势（墨绿）
```javascript
gradient.addColorStop(0, 'rgba(56, 142, 60, 0.3)');
gradient.addColorStop(0.5, 'rgba(56, 142, 60, 0.1)');
gradient.addColorStop(1, 'rgba(56, 142, 60, 0)');
```

## 快速示例

### 涨跌文字
```html
<text class="{{value > 0 ? 'rise' : value < 0 ? 'fall' : ''}}">
  {{value > 0 ? '+' : ''}}{{value}}%
</text>
```

### 金融数字
```html
<text class="finance-number">1.2345</text>
```

### 预警标签
```html
<text class="warning-badge">高溢价风险</text>
```
